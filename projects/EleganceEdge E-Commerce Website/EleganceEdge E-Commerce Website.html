EleganceEdge E-Commerce Website

Overview

Developed a fully functional e-commerce website, EleganceEdge, as part of a project. The platform allows users to browse products, add items to their cart, and place orders. It features a responsive design, dynamic content rendering, and secure user authentication.

Key Features

  • Product Management: Dynamic product listing and detailed product pages.
  • Shopping Cart: Interactive cart functionality with real-time updates using JavaScript and PHP.
  • User Authentication: Secure login and registration system with session management.
  • Order Placement: Confirmation page with order details and email notifications.
  • Responsive Design: Optimized for various screen sizes using CSS and media queries.

Technologies Used

PHP for backend development. JavaScript (with AJAX) for dynamic content updates. HTML5, CSS3 for responsive UI/UX. MySQL for database management. External libraries like FontAwesome and Slick Carousel for enhanced visuals.

Achievements

  • Successfully implemented a modular architecture with reusable components (e.g., header, footer).
  • Integrated a sliding product carousel for a modern user experience.
  • Designed and developed within a collaborative environment.
